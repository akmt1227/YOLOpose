# Pinned to the versions this project was developed/tested against.
#   uv venv --python 3.12 && uv pip install -r requirements.txt
#
# GPU build (CUDA 12.6) for Windows + Python 3.12 (cp312). cu126 wheels run on
# BOTH dev machines: NVIDIA drivers are backward compatible, so the RTX 4070 box
# (newer driver) runs them fine, while the RTX 4050 laptop's driver (566.14,
# CUDA <= 12.7) cannot load cu130 wheels — hence cu126, not cu130.
# torch/torchvision are pinned to the exact CUDA wheels via direct URLs so the
# plain `uv pip install -r requirements.txt` above resolves them with no extra flags.
#
# For a DIFFERENT Python/OS, swap the two URLs to the matching wheel from
# https://download.pytorch.org/whl/cu126/ (cp312->cpXY, win_amd64->your platform), or
# use the portable index form instead of the two torch/torchvision lines below:
#   --extra-index-url https://download.pytorch.org/whl/cu126
#   torch==2.13.0+cu126
#   torchvision==0.28.0+cu126
#   # and run with: uv pip install -r requirements.txt --index-strategy unsafe-best-match
#   # (needed: numpy is mirrored on the CUDA index at a different version)
#
# For CPU-only: replace the two URLs with  torch==2.13.0  and  torchvision==0.28.0
